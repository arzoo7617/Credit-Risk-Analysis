
# 🚀 Credit Risk Analysis

## 📌 Overview

This project implements a **Credit Risk Scorecard** using **Logistic Regression** to estimate the **Probability of Default (PD)** for loan applicants. The system supports underwriting decisions by analyzing borrower risk factors and generating data-driven loan approval recommendations, risk grades, and pricing insights.

---

## 🎯 Key Features

### 1. Probability of Default (PD) Prediction

* Estimates the likelihood of a borrower defaulting on a loan.
* Uses a Logistic Regression-based scorecard model.

### 2. Credit Grading System

* Automatically assigns credit grades (**A–E**) based on risk level.
* Simplifies applicant risk classification.

### 3. Underwriting Recommendations

* Generates **Approve** or **Decline** decisions.
* Supports automated credit assessment workflows.

### 4. Risk-Based Loan Pricing

* Calculates risk-adjusted **Annual Percentage Rate (APR)**.
* Aligns loan pricing with borrower risk profiles.

### 5. Financial Calculations

* Computes:

  * Monthly EMI
  * Total Interest Payable
  * Debt-to-Income Ratio (DTI)
  * Loan-to-Income Ratio (LTI)

### 6. Explainable Risk Analytics

* Identifies factors contributing to default probability.
* Improves transparency in lending decisions.

### 7. Interactive Power BI Dashboard

* Visualizes applicant risk profiles and portfolio performance.
* Enables data-driven decision-making.
